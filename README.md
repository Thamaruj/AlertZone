**AlertZone** is a cross-platform solution designed to bridge the communication gap between citizens and municipal authorities in Sri Lanka. ## Problem Statement

In Sri Lanka, non-emergency safety issues often go unreported because traditional reporting methods are time-consuming and lack transparency.

This results in a reactive governance model where hazards are addressed only after accidents occur.

AlertZone introduces a preventive approach through:
- Community-driven issue identification  
- Real-time tracking  
- Transparent communication between citizens and authorities
